# Knowledge Base Instructions

You are working inside the foundational knowledge base.

## Structure

| Folder | Content |
|--------|--------|
| `.apm/knowledge/constitution/` | Core engineering principles |
| `.apm/knowledge/governance/` | Architecture, security, testing guardrails |
| `.apm/knowledge/playbooks/` | Step-by-step delivery and workflow playbooks |
| `.apm/knowledge/brand/` | Brand guidelines and references |

## Editing Rules

- Keep entries concise and opinionated — these are guardrails, not essays.
- Prefer bullets over prose.
- Each file should cover one concern.
- These files are referenced by agents and skills — changes affect behavior.
