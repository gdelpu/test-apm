Project Booster

Project Booster exists to lower the entrance barrier of DevSecOps and cloud-ready applications, allowing you to get started effortlessly and without requiring a wide scope of knowledge.

Choose a scenario, follow the guided process and let Project Booster do the heavy lifting. You can learn new technologies and practices by implementing them now and understand them as you use them.

Depending on your needs and constraints, you can either use:

  • Project Booster public cloud instance
  • Project Booster private cloud instance
