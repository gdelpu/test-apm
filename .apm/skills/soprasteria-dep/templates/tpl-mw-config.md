## 6. Activation guide

1. Download Modern Workstation from the DEP portal or via the setup script.
2. Copy `mw-config.yml` to your working directory.
3. Run `mwctl module configure <module>` for each module in the config, in order.
4. Verify each module is operational (check with the `start` command for services).
5. Share `mw-config.yml` in the project repository under `docs/dep/` for team reuse.

> **Corporate environment tip:** for modules that require certificates (Git, Java TLS, etc.) set `corporate_cert` in the module configuration block.
