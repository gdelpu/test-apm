# Skill: Sopra Steria Document Branding

## Purpose
Use this skill to adapt PowerPoint, Word, PDF, proposal, dossier, memo, brochure, and similar assets to Sopra Steria branding.

## Priority order
1. use official template if available
2. use official logo and icon assets
3. correct typography and hierarchy
4. align layout to the composition system
5. verify accessibility and consistency

## PowerPoint guidance
- Prefer the official corporate or sector-specific PowerPoint templates.
- Supported formats mentioned in the guideline include 16:9 and A4 corporate templates.
- Use official layouts instead of manually reproducing master slides.
- Keep covers aligned with the horizontal layout system.
- Use brand icons from the official icon gallery.
- Do not overload slides with gradients; keep the look airy and structured.
- Favor visual hierarchy with white space, strong headings, and restrained accent color.

## Word guidance
- Prefer official Word templates.
- Available template families mentioned in the guideline include memo, letter, file, and CV templates.
- Keep A4 as the reference format.
- Body text in office documents should use Tahoma.
- Main headings can follow the brand heading approach while preserving office practicality.

## Cover / title page rules
For vertical covers:
- logo width ≈ 1/4 of document width
- title margins = 3X left and right

For horizontal covers:
- logo width ≈ 1/3 of document height
- title margins = 2X left and right

Where `X` is derived from the width of the logo swirls.

## Typography rules
- Brand font: Hurme Geometric Sans 3
- Main heading support font: Hurme Geometric Sans 4
- Office body text: Tahoma
- Keep line lengths comfortable
- Prefer left alignment except specific justified running text or centered quotes

## Icon usage
- Use the approved icon library where possible.
- Icons should remain outline-based and dark purple.
- If using color patches, keep them smaller than the icon and limited to approved style.

## Co-branding in documents/slides
- Place a gradient separator line between logos.
- Position Sopra Steria left if Sopra Steria initiated the communication, otherwise right.
- Balance logo sizes visually.

## Email / campaign note
Mass email campaigns should use dedicated tools and approved templates rather than ad hoc hand-crafted layouts.

## Deliverable format for document requests
When responding, provide:
1. asset type and template recommendation
2. slide/page-level issues
3. concrete corrections
4. optional rewritten layout structure
5. asset dependencies still needed
