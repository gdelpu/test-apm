# Create a Branded One-Pager

Create a concise one-pager in Markdown following Sopra Steria brand guidelines.

## Requirements

- Use H1/H2/H3 with heading font (Hurme Geometric Sans 4; fallback Tahoma).
- Body text in Hurme Geometric Sans 3 (fallback Tahoma/Segoe UI/Arial).
- Primary palette only for accents (links, callouts).
- Include 2–4 outline icons in dark purple (`#4D1D82`).
- Color patches limited to 20% opacity, smaller than icon.
- Meet AA contrast — avoid low-contrast pairings; do not rely on color alone.
- Airy layout (white background, minimal saturated blocks).

## Output

Save as `docs/<slug>.md`.
