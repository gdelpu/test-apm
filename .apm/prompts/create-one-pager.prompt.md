# Create a branded one‑pager from notes

Create a concise one‑pager in Markdown that:
- Uses H1/H2/H3 with heading font feel (Hurme 4; fallback Tahoma) and body in Hurme 3/Tahoma.
- Uses primary palette only for accents (links, callouts).
- Includes 2–4 outline icons in dark purple; if using color patches, limit to 20% opacity and smaller than icon.
- Meets AA contrast: avoid low‑contrast pairings; do not rely on color alone.
- Keeps the layout airy (white background, minimal saturated blocks).

Save as `docs/<slug>.md`. Use the `brand-document` skill for DOCX/PDF conversion if needed.
