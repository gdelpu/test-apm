# Sopra Steria Brand Audit

Assess an application, document, slide deck, page, mockup, or codebase against Sopra Steria branding and produce a structured compliance report.

## What to Inspect

1. Logo usage and placement
2. Color palette and gradient usage
3. Typography choices and hierarchy
4. Layout grid, spacing, and white-space
5. Images and iconography
6. Office-template compliance (Word/PPT)
7. Accessibility and contrast
8. Naming/casing consistency for Sopra Steria
9. Web accessibility (WCAG 2.1 AA) when target is a web application

## Output Format

### 1. Executive Summary
- Overall compliance score /100
- Key strengths
- Key issues

### 2. Detailed Findings
Table with columns: Area, Finding, Severity, Recommendation

### 3. Remediation Roadmap
Prioritized list of changes ordered by impact and effort.
