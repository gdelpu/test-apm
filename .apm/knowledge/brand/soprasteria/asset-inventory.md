# Sopra Steria Brand Assets Inventory

## Purpose
This inventory lists all official Sopra Steria branding resources available in this repository.

Agents and developers must always use these official assets when applying Sopra Steria branding to:

- applications
- documents
- presentations
- design systems
- marketing materials

Generated or approximated branding elements must **never replace official assets**.

# 2. Logos

Location

SOPRASTERIA_logo_RVB_noir_exe.png (black logo)  
SOPRASTERIA_logo_RVB_blanc_exe.png (white logo)  

Description

Primary Sopra Steria logo variants.

Usage rules

Agents must:

- preserve logo proportions
- never recolor the logo
- never crop or distort the logo
- select the variant based on background contrast

Guidelines

Color logo → default usage  
Black logo → light backgrounds  
White logo → dark backgrounds

# Brand Compliance Priority

When applying Sopra Steria branding, agents must follow this priority order:

1. Official templates
2. Official logos
3. Official icon library
4. Brand guideline rules

If conflicts occur, the **Brand Guideline PDF is the authoritative source**.
